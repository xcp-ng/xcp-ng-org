# Glossary

All terms and acronyms.

* CoW: [Copy-on-write](https://en.wikipedia.org/wiki/Copy-on-write)
* Dom0: Abbreviation for "[domain 0](https://wiki.xenproject.org/wiki/Dom0)"
* DomU: In Xen terminology, a DomU (user domain) is what XCP-ng users would typically call a guest VM: a regular virtual machine running under the control of Dom0.
* DRBD: [Distributed Replicated Block Device](/xostor/#drdb)
* HVM: [Hardware Virtual Machine](https://wiki.xenproject.org/wiki/Understanding_the_Virtualization_Spectrum#Xen_Project_and_Full_Virtualization)
* LUN: [Logical Unit Number](https://en.wikipedia.org/wiki/Logical_unit_number)
* LVM: [Logical Volume Manager](https://en.wikipedia.org/wiki/Logical_Volume_Manager_(Linux)) ([manpage]([](https://www.man7.org/linux/man-pages/man8/lvm.8.html#DESCRIPTION)))
* NBD: [Network Block Device](https://en.wikipedia.org/wiki/Network_block_device) ([official doc](https://github.com/NetworkBlockDevice/nbd/blob/master/doc/proto.md))
* PBD: [Physical Block Device](/storage/#-how-to-modify-an-existing-sr-connection)
* PIF: [Physical Interface](/project/architecture/#pifs)
* PV: [Para-Virtualization](https://wiki.xenproject.org/wiki/Paravirtualization_(PV))
* QCOW: [_QEMU Copy On Write_](https://en.wikipedia.org/wiki/Qcow), a disk image file format ([specification](https://www.qemu.org/docs/master/interop/qcow2.html))
* SMAPI: [Storage Manager API](https://github.com/xapi-project/xapi-storage)
* SR: [Storage Repository](/storage/)
* VBD: [Virtual Block Device](https://docs.kernel.org/block/ublk.html)
* VG:  [LVM's Volume Group](https://www.man7.org/linux/man-pages/man8/lvm.8.html#DESCRIPTION)
* VHD: [_Virtual Hard Disk_ file format](https://en.wikipedia.org/wiki/VHD_(file_format)) ([specification](https://www.microsoft.com/en-us/download/details.aspx?id=23850))
* VMDK/VMDKX: [_Virtual Machine Disk_ file format](https://en.wikipedia.org/wiki/VMDK) ([specification](https://github.com/libyal/libvmdk/blob/main/documentation/VMWare%20Virtual%20Disk%20Format%20(VMDK).asciidoc))
* VM: [Virtual Machine](https://en.wikipedia.org/wiki/Virtual_machine), also called <abbr title="Unprivileged Domain">DomU</abbr> or _guest_
* XAPI: [Xen Project Management API](https://github.com/xapi-project/xen-api)
* XCP-ng: [XCP New Generation](https://xcp-ng.org/)
* XCP: [Xen Cloud Platform](https://wiki.xenproject.org/wiki/XCP_Overview) (legacy Citrix project)
* XOA: [Xen Orchestra Appliance](/management/manage-at-scale/xo-web-ui/)
* XOSTOR: [Hyper-convergence solution based on LINSTOR](/xostor/)
